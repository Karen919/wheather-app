# Weather App

Weather app using [Openweahtermap API](https://openweathermap.org/api) and [IP Geolocation API](https://geo.ipify.org)

### Built with

- Semantic HTML5 markup
- CSS custom properties
- Flexbox
- JavaScript
- Mobile-first workflow
